## Claude / Claude Code
- **Claude Code 実践ガイド【2026年4月版】** — Codex・Claude Code・GitHub Copilotを「適材適所」で使い分ける方法が解説されています。Claude Codeはアーキテクチャ設計・大規模リファクタリングに強み。（ソース: [Zenn](https://zenn.dev/motowo/articles/codex-claude-code-copilot-2026)）
- **知らないと損！Claude Code 便利コマンド**
  - `/btw` — 会話履歴を汚さずに小さな補足コンテキストを渡せるコマンド
  - `/loop` — プロンプトを指定間隔で繰り返し実行（例: 5分ごとにテスト実行）
  - `/simplify` — コードを並列サブエージェントが自動レビュー・改善（2026年2月28日追加）
  - `/compact` — 会話履歴を要約し、`/clear`より穏やかなリセット
  - `#` から始める入力でルールをauto-memoryに自動保存
  （ソース: [Zenn 全機能リファレンス](https://zenn.dev/sun_asterisk/articles/38be67ca250159) / [Qiita](https://qiita.com/nogataka/items/5e64037cc452c5d497fa)）
- **Claude Codeステータスライン完全ガイド【2026年最新版】** — ステータスラインのカスタマイズから自作ツール開発まで解説。（ソース: [Qiita](https://qiita.com/dai_chi/items/d72ec42444d66e88a044)）
- **【注意】Claude Code セキュリティバグ** — Deny Rulesが「51個目のコマンド」で無効化される致命的バグが報告されています。設定したdeny rulesが機能しなくなる可能性があるため注意が必要。（ソース: [Qiita](https://qiita.com/emi_ndk/items/b6f8969da1b833d764aa)）
- **Claude Codeでのコード自動生成率** — AnthropicはClaudeが一部プロジェクトのコードの最大90%を自律的に執筆すると発表。AI主導のソフトウェア開発が現実に。（ソース: [crescendo.ai](https://www.crescendo.ai/news/latest-ai-news-and-updates)）
